# Backend & Automation
- **PowerShell:** Core automation for interacting with the Azure DevOps REST API.
- **Python:** Pre-processing utilities (e.g., .vtt cleaning, .docx to .txt conversion).

# Logic & Data
- **Markdown:** Master prompts for guiding LLMs in synthesis and backlog generation.
- **JSON:** Primary format for intermediate artifacts (synthesis results) and final backlog data.
- **Mermaid:** Used for creating flowcharts, sequence diagrams, and state diagrams in documentation.

# External Platforms
- **Azure DevOps:** Target platform for task and user story creation.
- **OpenAI/Gemini (LLM):** External AI engines used for requirements processing.
